About Project

Recently I got into cooking at home and thought about having an app that can provide simple recipes for dishes which can massively minimise time looking for something to cook. For this project I had to use a Spoonacular API to retrieve recipes and information on how to prepare. Overall this project alllowed me to practice using APIs in more real-life scenarios.
